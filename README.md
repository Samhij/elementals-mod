# Elementals Mod

Elementals Mod is a Minecraft Fabric mod for 1.21.1 that introduces elemental-themed content to the game. It features custom items, blocks, equipment, and a unique flight power system.

## Features

* **Elemental Content**: Adds a variety of custom blocks and items themed around elements.
* **New Equipment**:
    * Custom Tool sets using new materials.
    * Custom Armor sets with unique material properties.
* **Flight System**: A custom flight power mechanic complete with a dedicated HUD overlay.
* **Advancements**: Custom progression and advancements.

## Installation

To play this mod, you need to follow these steps:

1. **Install Java**: Ensure you have Java 21 installed.
2. **Install Fabric Loader**: Download and install the Fabric Loader (version 0.18.4 or higher) for Minecraft 1.21.1.
3. **Download the Mod**: Get the latest .jar file for the Elementals Mod.
4. **Download Fabric API**: Download the Fabric API for Minecraft 1.21.1 (version 0.116.7+1.21.1).
5. **Install Files**: Place both the Elementals Mod .jar and the Fabric API .jar into your Minecraft `mods` folder.
6. **Launch**: Start the game using the Fabric profile in your Minecraft launcher.
